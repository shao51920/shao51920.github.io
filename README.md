# 🖼️ 图片网格裁剪工具

一个功能强大的在线图片裁剪工具，支持自定义网格分割和表情包批量转换。

## 🚀 使用方法
点击网址：https://shao51920.github.io/

对你有用的话，麻烦点个star⭐

## ✨ 功能特点

### 📤 图片上传
- 支持点击上传或拖拽上传
- 支持 JPG、PNG、GIF、WebP 等常见图片格式
- 即时预览上传的图片

### 📐 自由网格分割
- **独立设置行列数**：支持 1-10 行/列，可设置如 3×4、4×3、2×5 等任意组合
- **可拖动网格线**：
  - 🔴 红色横线 - 拖动调整行高
  - 🔵 蓝色竖线 - 拖动调整列宽
  - 拖动时高亮显示，支持触摸屏操作
- **一键重置**：快速恢复均匀分布的网格线
- 实时显示裁剪后的图片总数

### ✂️ 智能裁剪
- 根据自定义网格线位置精确裁剪
- 裁剪结果按网格排列展示
- 每张图片显示位置标记（如 1-1、1-2...）
- 支持单张下载或批量下载全部

### 😊 表情包转换
裁剪完成后可将图片批量转换为表情包格式：

**格式选择：**
- PNG（透明背景）
- GIF
- WebP（推荐，文件小质量高）
- JPEG

**尺寸预设：**
- 64×64（小图标）
- 128×128（标准表情）
- 240×240（微信表情包）
- 300×300（QQ 表情包）
- 512×512（大尺寸）
- 自定义尺寸（16-1024px）


## 🔒 隐私安全

- ✅ 所有图片处理均在浏览器本地完成
- ✅ 不会上传任何图片到服务器
- ✅ 无需注册登录
- ✅ 关闭页面后数据自动清除
