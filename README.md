# 觉醒诗社 · Awakening Poetry

> “觉醒是一场从内部燃起的革命。”

欢迎来到**觉醒诗社**。这是一个致力于生命真相探索、意识进化与自我解剖的精神观测站。在这里，我们抛却俗世标签，通过一系列精心设计的工具，助你照见最真实的自己。

## 🌌 核心模块

### 1. [灵性人格测试 (SoulLab)](https://qingye520.xyz/SoulLab/)
抛却 MBTI 的刻板定义，用 33 个直指内心的叩问，带你进行一次温柔而彻底的「自我解剖」。
- **特色**：12 种独特的灵性原型显化，深空呼吸感的视觉交互。

### 2. [自我客体化测评 (Objectification)](https://qingye520.xyz/Objectification/)
评估你在多大程度上将自己当作工具、物品或他人实现目的的手段。
- **功能**：40 道专业测评题，详细的心理状态分析与行动建议。

### 3. [觉醒智库](https://ncnra455tpou.feishu.cn/wiki/R4y5wdFngiviwWkz1uQcYSDInmh)
汇集关于意识、觉知、真理与自由的结构化知识库，为你提供觉醒之路上的理论支撑。

### 4. [在线赏雪 (Snow)](https://qingye520.xyz/Snow/)
一个沉浸式的 3D 漫天飞雪场景，支持视角拖拽与参数调节。
- **意境**：在纷飞的白雪中找回内心的静谧。
