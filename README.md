# 🖼️ 图片网格裁剪工具

一个功能强大的在线图片裁剪工具，支持自定义网格分割和表情包批量转换。

## ✨ 功能特点

### 📤 图片上传
- 支持点击上传或拖拽上传
- 支持 JPG、PNG、GIF、WebP 等常见图片格式
- 即时预览上传的图片

### 📐 自由网格分割
- **独立设置行列数**：支持 1-10 行/列，可设置如 3×4、4×3、2×5 等任意组合
- **可拖动网格线**：
  - 🔴 红色横线 - 拖动调整行高
  - 🔵 蓝色竖线 - 拖动调整列宽
  - 拖动时高亮显示，支持触摸屏操作
- **一键重置**：快速恢复均匀分布的网格线
- 实时显示裁剪后的图片总数

### ✂️ 智能裁剪
- 根据自定义网格线位置精确裁剪
- 裁剪结果按网格排列展示
- 每张图片显示位置标记（如 1-1、1-2...）
- 支持单张下载或批量下载全部

### 😊 表情包转换
裁剪完成后可将图片批量转换为表情包格式：

**格式选择：**
- PNG（透明背景）
- GIF
- WebP（推荐，文件小质量高）
- JPEG

**尺寸预设：**
- 64×64（小图标）
- 128×128（标准表情）
- 240×240（微信表情包）
- 300×300（QQ 表情包）
- 512×512（大尺寸）
- 自定义尺寸（16-1024px）

**智能处理：**
- 自动缩放至正方形
- 保持原图比例居中
- 透明背景支持（PNG/WebP/GIF）
- 进度条实时显示转换状态

## 🚀 使用方法

1. **上传图片**
   - 点击上传区域选择图片，或直接拖拽图片到上传区域

2. **设置网格**
   - 使用 +/- 按钮调整行数和列数
   - 拖动红色横线调整每行高度
   - 拖动蓝色竖线调整每列宽度
   - 如需重新调整，点击"重置网格"恢复均匀分布

3. **裁剪图片**
   - 点击"开始裁剪"按钮
   - 查看裁剪结果预览
   - 点击单张图片下载，或点击"下载全部"批量下载

4. **转换表情包**（可选）
   - 选择输出格式（推荐 PNG 或 WebP）
   - 选择表情包尺寸
   - 点击"批量转换表情包"
   - 等待转换完成后下载
